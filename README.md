# Sales Leaderboard - Ironbridge Wholesale Company

Multi-user sales leaderboard for tracking team performance with weekly/monthly/yearly views.

## Quick Deploy to Vercel

1. Upload this folder to GitHub (or use Vercel CLI)
2. Go to https://vercel.com/new
3. Import this repository
4. Click Deploy
5. Done! Your app will be live at https://your-project.vercel.app

## Local Development

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Setup Required

Before first use, run the SQL schema in your Supabase SQL Editor.
See supabase_schema.sql for the schema.
